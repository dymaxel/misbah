Sales Agent Select on Commission Shop

Commission Sales Agent on Sales Order Shop

This app allows your customer to set a sales agent / external agent while purchasing from your website shop. When sales order will be generated from website shop order by customer then system will automatically set Sales agent on commission and same time if you have set salesperson configuration on customer for website order then system will also consider and add sales person in commission list.

Main Features:

  • Allow your customer to select sales agent on the sales order shop.
  • It will set the sales agent and salesperson to the sales order for commission.
  • For more details see below screenshots.

Sales Commission Level Configuration

Configure Sales Commission Level for Shop Sales Agent.

Sales Commission Level Configuration

    There is two types of configuration on sales commission level
  • - Use Webshop Sales Agent
    • When Checked this option will consider level as sales agent when customer add agent from shop payment form (Only for Sales order generated by Website Shop)
  • - Use Webshop Sales Person
    • When Checked this option will consider level as sales person when customer add agent from shop payment form (Only for Sales order generated by Website Shop)

Customer form configuration for sales agents and salespersons

  • - Is Sales Agent
    • If this checkbox is checked this person will be listed as a sales agent on a website. (Only for Sales order generated by Website Shop)
  • - Website Order Sales Person
    • If this checkbox is checked then customer sales order commission will be applicable to the sales person of this customer also. (Only for Sales order generated by Website Shop)
